## UBER EATS Clone Backend

The backend for the UBER Eats Clone project. Built with NestJS.


## User Model:

- id
- createdAt
- updatedAt

- email
- password
- role(client|owner|delivery)

## User CRUD:

- Create Account
- Log In
- See Profile
- Edit Profile
- Verify Email

## Restaurant Model:
- name
- category
- address
- coverImage
